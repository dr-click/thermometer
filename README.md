## Project Details:

* Environment

  > API Rails Project

  > Ruby 2.6.1

  > Rails 5.2.2

  > PostgreSQL

  > RSpec

* Configuration

  > copy `config/database.yml` from `config/database.yml.example`

* Database creation

  > run `bundle exec rake db:create`

  > run `bundle exec rake db:migrate`

  > run `bundle exec rake db:seed`


## Production Deploy and Recommendations:

  > copy `config/database.yml` from `config/database.yml.example`
